Web Development 701 ~ Comparison of frameworks and CMS

Todays class covers a comparison of frameworks and CMS. The tutor gave access to an online document with the following questions. We split up into groups, so I will review the answers my group (Bryce, Finn, Deja, and myself).

What is a CMS?

A platform that allows non-developers to manage the content of a website

How many are there?

Over 9000

Which are the most popular?

Wordpress

List the top ten?

Wordpress, Joomla, Drupal, Magento, Blogger, Shopify, TYPO3, Bitrix, Squarespace, PrestaShop

Note This top ten list was resourced from “websitesetup.org/popular-cms”

Which is best?

Depends on the requirements and the team

Why?

One should always choose the right tool not use one for all purposes. The same applies here.

On which platforms are they?

Webservers

How many claim to be a web framework?

None

What is the difference between a framework and a CMS?

Frameworks are built to make development easier. CMS are also built to make development easier, but a CMS is built on top of a framework.

How many web frameworks are there?

Over 9000

How many CMSs’?

Indeterminable
