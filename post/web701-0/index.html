Web Development 701

Web development is a required class for my major (software development) in the bachelor of IT. The 701 class is based around information architecture, CMS (content management systems), frameworks and website security. A CMS is essentially a platform which allows a user to change the website without touching any code, usually the form of drag and drops. I don’t program using PHP at all, and don’t really have any interest in web development so I am going to see if I can use Odoo, since it has a website module that is a CMS. After conversing with the tutor he said I can use whatever framework/cms I want for the individual assessment, but wants me to install laravel and meteor for practice (which is fair).

I missed the first day due to being out of town, but by the looks of it the only thing I missed was the start of the practice assessment. Fortunately I know one of the others in the class, Bryce, who invited me to join his team. Our team’s name is Team Chillen Oxen and is comprised of 5 people; Finn, Stephen, Bryce, Deja, and myself. I am grateful to these lads as they could have easily ignored me when I was sitting in the student lounge by myself, not knowing what was going on for web yet.
